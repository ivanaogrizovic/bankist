## About The Project

Bankist is a front-end JavaScript project that showcases the layout and structure of a digital banking interface. It focuses on DOM manipulation and UI interactions to demonstrate how features like transactions and account activity can be presented in the browser.<br>
This project serves as a learning exercise and visual prototype rather than a fully functional application.
